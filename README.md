# GrowIndiaUP - Corporate Website

A professional corporate website for GrowIndiaUP, a premier building glass manufacturer offering investment opportunities.

![Website Screenshot](https://github.com/user-attachments/assets/47a6460b-db63-4689-a987-fb88f20289af)

## Project Overview

This is a single-page responsive website showcasing GrowIndiaUP's manufacturing capabilities, investment plans, and company information. The website is built using vanilla HTML, CSS, and JavaScript without any frameworks or dependencies.

## Features

### ‚úÖ Completed Features

- **Responsive Design**: Fully responsive layout that works seamlessly on desktop, tablet, and mobile devices
- **Navigation**: 
  - Smooth scrolling to different sections
  - Mobile-friendly hamburger menu with accessibility features (ARIA labels)
  - Sticky header for easy navigation
- **Sections**:
  - Hero section with compelling call-to-action
  - About section highlighting company legacy and innovation
  - Manufacturing process showcase (3-stage process)
  - Partner testimonials
  - Investment plans (6 different plans: Star, Silver, Gold, Platinum, Ultimate, and Infinite)
  - Why Choose Us section with 6 key benefits
  - Contact form with client-side validation
  - Footer with quick links, legal links, and newsletter subscription
- **Animations**: Intersection Observer-based scroll animations for enhanced user experience
- **Contact Form**: Functional form with success message display
- **Images**: Optimized WebP images for faster loading
- **Accessibility**: ARIA labels on interactive elements

## Technology Stack

- **HTML5**: Semantic markup for better SEO and accessibility
- **CSS3**: Modern styling with flexbox, grid, and animations
- **JavaScript (ES6+)**: Vanilla JavaScript for interactions
  - Form handling
  - Hamburger menu toggle
  - Scroll animations with Intersection Observer API

## How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/code-with-mahir/Corporate-Website.git
   ```

2. Navigate to the directory:
   ```bash
   cd Corporate-Website
   ```

3. Open `index.html` in your browser or use a local server:
   ```bash
   # Using Python
   python3 -m http.server 8080
   
   # Using Node.js
   npx serve
   ```

4. Visit `http://localhost:8080` in your browser
